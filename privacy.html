Privacy Policy

WhatsApp Privacy Plus is a Chrome extension designed to enhance privacy on WhatsApp Web by visually blurring sensitive interface elements.

Data Collection

This extension does not collect, store, transmit, or share any personal data.

Local Storage

The extension stores user preferences locally using Chrome’s storage API, including:

  • Blur settings
  • Privacy mode state
  • User configuration preferences

Data Sharing

No data is shared with third parties or external servers.

Remote Code

This extension does not use remote code or external scripts. All functionality is included within the extension package.
