# Privacy Policy for WhatsApp Privacy Plus

Last Updated: June 2026

## Overview

WhatsApp Privacy Plus is designed to enhance user privacy while using WhatsApp Web by applying visual blur effects to selected interface elements.

This extension operates entirely within the user's browser and does not collect, store, transmit, or sell personal information.

## Information Collection

WhatsApp Privacy Plus does not collect any personal information.

The extension does not collect:

* Names
* Email addresses
* Phone numbers
* Chat messages
* Contact information
* Authentication credentials
* Location information
* Browsing history
* Financial information

## Data Storage

The extension stores user preferences locally using Chrome's storage API.

Examples of stored settings include:

* Privacy mode status
* Blur intensity preferences
* Selected blur targets
* Extension configuration settings

These settings remain on the user's device and are not transmitted to external servers.

## Data Sharing

WhatsApp Privacy Plus does not:

* Sell user data
* Share user data with third parties
* Transfer user data externally
* Use user data for advertising purposes

## Remote Code

The extension does not download or execute remote code. All functionality is packaged within the extension.

## Third-Party Services

WhatsApp Privacy Plus does not use analytics services, advertising services, tracking services, or external APIs.

## Security

Because the extension does not collect or transmit personal information, user data remains under the user's control within their browser environment.

This Privacy Policy may be updated periodically. Any updates will be reflected on this page with a revised "Last Updated" date.
